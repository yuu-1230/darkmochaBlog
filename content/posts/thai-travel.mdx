もう1年半前のことだけど、せっかくブログ作ったのでタイ旅行のことを記録してみます。  
大学１年生の夏休み、人生初めての海外旅行・一人旅をしました。  
きっかけは、父に勧められたところから。  
一人は少し怖かったけど、自分のペースで好きな場所・食べ物・経験が出来た良い思い出。  
結構贅沢した旅行だった気がします。

## 予算

旅行したとき（2024/9）でレートは**約4.2円**。  
1年半で5円まで上がってしまいました。  
円安が進んでタイの物価が上がってしまった印象です。  
宿泊費は日本に比べてとても安く **一泊3000～** 泊まることが出来ます。  
移動はタクシー（Grab）、BTSなどの電車を利用しました。  
ドミトリーを使ったり上手く計画すればもっと節約した旅行が出来ます。

| 項目                 | 金額                        |
| -------------------- | --------------------------- |
| レート               | 1タイバーツ（THB）= 約4.3円 |
| 交通費               | 53,000円                    |
| 宿泊費               | 36,000円                    |
| 観光費               | 24,000円                    |
| 食費                 | 13,000円                    |
| その他（お土産など） | 20,000円                    |
| **合計**             | **146,000円**               |

## Day1 日本からタイへ

### 成田空港〜スワンナプーム空港

ANAグループの第三のブランドであるLCC、AirJapanに乗ってスワンナプーム国際空港へ。  
自分の記録が正しければ、**往復航空券24400円**で行けた。破格すぎる...

## Day2 ウィークエンドマーケット

### チャトゥチャック・ウィークエンド・マーケット

とにかく広くて、全部見るのは不可能に近い巨大市場。
自分へのお土産にタイパンツを買ったり、歩き疲れたらココナッツアイスを食べたり。

## Day3 ツアー（水上マーケット・アユタヤ）

### メークロン線路市場

電車が通るギリギリまで市場が広がっている場所。
汽笛が鳴ると一斉に屋根が畳まれる光景は、タイに来た実感が一番湧いた瞬間でした。

### アユタヤ遺跡

夕暮れ時の遺跡はとても神秘的。
歴史の重みを感じながら、ゆっくりと散策しました。

## Day4 マハナコンタワー・カオサンロード

### キングパワー・マハナコン

バンコクで一番高いビルの展望台。
ガラス床の上を歩くのは足がすくみましたが、360度の絶景は一生の思い出。

### カオサンロード

「バックパッカーの聖地」と呼ばれるだけあって、夜の熱気がすごかったです。
ここで食べた屋台のパッタイが一番安くて美味しかった。

## Day5 水上ホテル in ジャングル

### リバーサイドの隠れ家ホテル

バンコクの喧騒を離れて、川沿いのホテルへ。
自然の音を聞きながら、何もしない贅沢な時間を過ごしました。

## Day6 バンコク三大寺院・アジアンティークマーケット

### 三大寺院（ワット・アルン等）

朝から渡し船に乗って寺院巡り。
特にワット・アルンの陶器の装飾は、近くで見ると本当に繊細で綺麗でした。

### アジアンティーク・ザ・リバーフロント

最後の方の夜は少し贅沢に。
夜景を見ながらのディナーで、旅の思い出を振り返りました。

## Day7 タイから日本へ

### 帰国

あっという間の7泊9日。
初めての一人旅をタイにして正解でした。またいつか再訪したいです。
